# Kotlin + Springboot Reactive Example

### Description

This is example code with kotlin and springboot reactive stack(webflux).

### Environment

* Java 17
* Kotlin 1.8
* Springboot 3.1.4
* Gradle 8.3

### Features

* RestController with annotation
* Filter
* BaseResponse
* GlobalErrorAttributes
* NoSQL(Redis)
* Junit
* Testcontainers
